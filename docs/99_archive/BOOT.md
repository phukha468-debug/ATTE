# ATTE (66AI Attestation) — BOOT

> **Telegram Mini App для B2B-аттестации ИИ-компетенций**
> Статус: Этап 2 завершен, переход к Этапу 3.

---

## 🚀 Текущий статус проекта

### ✅ Реализовано (Этапы 1-2):
- **Фундамент:** Telegram Auth (HMAC) + Supabase (RLS).
- **Этап 1 (Аттестация):** Движок тестов, загрузка вопросов, оценка через OpenRouter, сохранение результатов.
- **Этап 2 (Симулятор):** 
  - Разветвленная система выбора (11 направлений, 22 роли).
  - Интерфейс песочницы (Sandbox) с ИИ-ассистентом (Gemini 2.0 Flash).
  - LLM-Судья (Claude 3.7 Sonnet) с оценкой навыков и коэффициентом ускорения.
  - Синхронизация прогресса с главной страницей и разделом аналитики.
- **UI:** Адаптивная верстка, компактный дизайн, поддержка мобильных устройств (dvh, safe-areas).

### ⏳ В планах (Этап 3):
- **HR-Дашборд:** Глубокая аналитика для руководителей (агрегация по компании).
- **Микро-проекты:** Пошаговые сценарии автоматизации реальных задач.
- **Монетизация:** Платежный шлюз для корпоративных тарифов.

---

## 🛠 Быстрый старт

1. **Установка зависимостей:** `npm install`
2. **Окружение:** Создайте `.env` на базе `.env.example`.
3. **Запуск:** `npm run dev` (Frontend на :3000)
4. **Backend:** Для работы API запустите `npx vercel dev` (на :3001).
