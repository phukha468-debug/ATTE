# МИКРО-ПРОЕКТ: База знаний для работы с жалобами
**Сотрудник:** Анна Сергеева | **Компания:** Логистик-Групп

## 1. СВЯЗЬ С ПРЕДЫДУЩИМИ ЭТАПАМИ
- **Этап 1:** Выявила рутину — ответы на вопросы о задержках грузов (10 часов в неделю).
- **Этап 2:** Балл за промптинг 70. Зона роста: эмпатия в ответах.
- **Выбранная задача:** Генератор вежливых ответов на жалобы по задержкам.

## 2. ОПИСАНИЕ РЕШЕНИЯ
**Инструмент:** Claude 3.5 Sonnet
**Пайплайн:** Создан системный промпт, который принимает номер заказа и причину задержки, а на выходе дает текст для мессенджера с признанием вины и бонусом для клиента.

## 3. ЭКОНОМИЯ (ROI)
- **Время вручную:** 10 минут на сложный ответ.
- **Время с ИИ:** 2 минуты.
- **Экономия:** 8 минут на тикет.
- **В неделю (50 тикетов):** ~6.5 часов экономии времени отдела.
