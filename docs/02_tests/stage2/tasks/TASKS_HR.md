# ПУЛ ЗАДАНИЙ: HR-МЕНЕДЖЕР / РЕКРУТЕР

> **Профиль:** hr | **Заданий:** 7 (2+3+2) | **Лимит:** 45 мин

---

## УРОВЕНЬ: БАЗОВЫЙ

### ЗАДАНИЕ B-1: Описание вакансии

**Контекст:** Нанимающий менеджер прислал заявку на подбор. Нужно написать описание вакансии для размещения на hh.ru и в Telegram-канале.

**Вводные данные:**
```
Компания: ООО «ТехноПарт», дистрибьютор компьютерной техники, Москва
Размер: 45 сотрудников
Культура: стабильная компания, 12 лет на рынке, гибрид 3/2

Вакансия: Бухгалтер (на замену уходящему в декрет сотруднику)
Требования от руководителя:
- Опыт от 1 года, знание 1С:Бухгалтерия
- Первичная документация, банк-клиент, отчётность
- Зарплата: 70-85 тыс. рублей
- ДМС через 3 месяца, 13-я зарплата
```

**Задание:** С помощью ИИ напишите описание вакансии с цепляющим заголовком, описанием компании, обязанностями, требованиями и условиями.

**Формат:** Готовый текст вакансии. **Бенчмарк:** 25 мин.

---

### ЗАДАНИЕ B-2: Письмо-отказ кандидату

**Контекст:** Вы провели собеседование. Кандидат не подошёл. Нужно написать корректный отказ.

**Вводные данные:**
```
Кандидат: Алексей, 26 лет
Позиция: менеджер по продажам (B2B, IT)
Собеседование: прошло хорошо по коммуникации, но:
- Опыт только B2C (розничные продажи электроники)
- Не знаком с длинным циклом B2B сделок
- Зарплатные ожидания выше вилки (хочет 180 тыс, вилка до 150)
Хотите: сохранить в резерве на позицию junior или если появится B2C-вакансия
```

**Задание:** С помощью ИИ напишите письмо-отказ. Честное, не обесценивающее, с предложением остаться в резерве.

**Формат:** Письмо 100-150 слов. **Бенчмарк:** 15 мин.

## УРОВЕНЬ: СРЕДНИЙ

### ЗАДАНИЕ M-1: Вакансия + скрипт собеседования

**Контекст:** Срочный подбор. Нужно и вакансию, и подготовиться к собеседованиям за один сеанс.

**Вводные данные:**
```
Компания: IT-интегратор, 80 сотрудников, Екатеринбург
Вакансия: Senior Frontend Developer (React)
Команда: 4 фронтендера, тимлид — Сергей (будет на техническом интервью)
Причина поиска: масштабирование продукта, новый клиент

Требования:
- React 3+ лет, TypeScript, Redux/MobX
- Опыт оптимизации производительности
- Code review, менторство junior
- Зарплата: 250-320 тыс. (на руки), удалёнка полная
- EVP: оплата конференций, 28 дней отпуска, stock options через 2 года
```

**Задание:** С помощью ИИ подготовьте:
1. Описание вакансии (для hh.ru + адаптация для LinkedIn)
2. Скрипт телефонного скрининга (10 вопросов с критериями оценки)

**Формат:** 2 документа. **Бенчмарк:** 75 мин.

---

### ЗАДАНИЕ M-2: Адаптационный план нового сотрудника

**Контекст:** Через неделю выходит новый менеджер по продажам. Нужен план адаптации.

**Вводные данные:**
```
Компания: SaaS-платформа для автоматизации складов, 60 чел., Москва
Новый сотрудник: Ирина, 30 лет, опыт B2B 4 года (не в IT)
Позиция: менеджер по продажам, отдел из 8 человек
Наставник: Дмитрий (старший менеджер, грейд 4)
Продукт: сложный, средний цикл сделки 2 месяца
Ожидание руководителя: первая самостоятельная сделка через 6-8 недель
```

**Задание:** С помощью ИИ подготовьте:
1. План адаптации на 90 дней (по неделям для первого месяца, по месяцам дальше)
2. Чек-лист для наставника (что передать, чему научить)
3. Метрики успешной адаптации (что проверяем на 30/60/90 день)

**Формат:** Структурированный документ (2-3 страницы). **Бенчмарк:** 60 мин.

---

### ЗАДАНИЕ M-3: Анализ exit-интервью и рекомендации

**Контекст:** За квартал из компании ушли 5 сотрудников. HR провёл exit-интервью. Нужен анализ.

**Вводные данные:**
```
Компания: маркетинговое агентство, 35 сотрудников

Exit-интервью (краткие выдержки):

Сотрудник 1 (дизайнер, 2 года): «Устал от переработок. Каждый проект — 
срочный. Просил повышение полгода, обещали и забыли. Ухожу на удалёнку 
в продуктовую компанию, +30% к зарплате.»

Сотрудник 2 (аккаунт-менеджер, 1.5 года): «Нет роста. Делаю одно и то же.
Просил перевод в стратегию — сказали "может быть". Нашёл позицию 
проектного менеджера в другом агентстве.»

Сотрудник 3 (копирайтер, 8 мес): «Ожидания не совпали. На собеседовании
обещали креативные проекты, а 80% работы — SEO-тексты. Руководитель
редко даёт обратную связь.»

Сотрудник 4 (таргетолог, 3 года): «Просто выгорел. Клиентов стало 
в 2 раза больше, команда не выросла. Зарплату подняли на 10%, 
но нагрузка выросла на 100%.»

Сотрудник 5 (HR-ассистент, 1 год): «Начальница микроменеджит. 
Каждый шаг нужно согласовывать. Чувствую себя ненужной.»
```

**Задание:** С помощью ИИ подготовьте:
1. Анализ: выделите паттерны (сгруппируйте причины увольнений)
2. Оценка рисков: кто ещё может уйти по тем же причинам
3. Рекомендации руководству (конкретные действия, не абстрактные)

**Формат:** Аналитическая записка для директора (1-2 страницы). **Бенчмарк:** 45 мин.

## УРОВЕНЬ: ПРОДВИНУТЫЙ

### ЗАДАНИЕ A-1: HR-аналитика — отчёт по текучести

**Контекст:** CEO попросил подготовить отчёт по текучести за год с анализом причин и рекомендациями.

**Вводные данные:**
```
Компания: сеть ветеринарных клиник, 120 сотрудников, 6 филиалов, Москва

Данные за 2025 год:
- Среднесписочная: 115 чел.
- Уволилось: 28 чел. (текучесть 24.3%)
- Из них: по собственному — 22, по инициативе — 6

По отделам:
- Ресепшн (30 чел.) — ушли 12 (40%)
- Ветеринары (40 чел.) — ушли 6 (15%)
- Администрация (20 чел.) — ушли 4 (20%)
- Маркетинг (10 чел.) — ушли 3 (30%)
- IT + бухгалтерия (15 чел.) — ушли 3 (20%)

Средний стаж уволившихся: 11 мес.
Стоимость найма: ~80 000 ₽ на позицию (рекрутинг + адаптация)

Основные причины (из exit-интервью, топ-3):
1. Зарплата ниже рынка (35%)
2. Сложный график / переработки (25%)
3. Конфликт с руководителем филиала (20%)
```

**Задание:** С помощью ИИ подготовьте:
1. Аналитический отчёт (текучесть по отделам, динамика, бенчмарк рынка)
2. Расчёт стоимости текучести за год
3. Приоритизированный план снижения текучести (что делать первым)
4. KPI для отслеживания (метрики + целевые значения)

**Формат:** Отчёт для CEO (2-3 страницы). **Бенчмарк:** 120 мин.

### ЗАДАНИЕ A-2: Система мотивации для отдела продаж

**Контекст:** РОП просит пересмотреть систему мотивации — текущая не работает.

**Вводные данные:**
```
Компания: B2B дистрибьютор промышленного оборудования, 70 чел.
Отдел продаж: 12 менеджеров + РОП
Средний цикл сделки: 3-4 месяца
Средний чек: 1.2 млн ₽

Текущая система:
- Оклад: 80 000 ₽
- Бонус: 3% от маржи закрытых сделок
- Нет порога и потолка
- Нет командного KPI

Проблемы:
- 3 «звезды» делают 60% продаж, остальные 9 — 40%
- Junior-менеджеры уходят через 4-6 месяцев (нет сделок = нет бонусов)
- Нет мотивации помогать коллегам (каждый за себя)
- РОП хочет: удержать звёзд + вырастить mid-уровень + командный дух

Бюджет ФОТ отдела: 2.5 млн ₽/мес (не может вырасти более чем на 10%)
```

**Задание:** С помощью ИИ разработайте:
1. Новую систему мотивации (оклад + личный KPI + командный KPI)
2. Грейды для менеджеров (junior → mid → senior → lead) с критериями перехода
3. Модель расчёта бонусов с примерами (3 сценария: слабый/средний/сильный месяц)
4. План перехода на новую систему (как внедрить без шока)

**Формат:** Документ (3-4 страницы) с таблицами и расчётами. **Бенчмарк:** 90 мин.
