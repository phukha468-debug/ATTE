# ПУЛ ЗАДАНИЙ: ОФИС-МЕНЕДЖЕР / АССИСТЕНТ РУКОВОДИТЕЛЯ

> **Профиль:** office | **Заданий:** 7 (2+3+2) | **Лимит:** 45 мин

---

## УРОВЕНЬ: БАЗОВЫЙ

### ЗАДАНИЕ B-1: Объявление для сотрудников

**Вводные данные:**
```
Компания: маркетинговое агентство, 40 сотрудников, гибрид
Канал: корпоративный Telegram-чат

Ситуация: с понедельника меняется Wi-Fi в офисе.
- Старая сеть «OfficeNet» перестанет работать
- Новая сеть: «Agency_5G» / пароль: Spring2026!
- Причина: обновление оборудования, скорость вырастет в 3 раза
- Принтер тоже переподключится, но автоматически
- Если что-то не работает — писать в чат #it-support или Диме (админу)
- Гостевая сеть для клиентов: «Agency_Guest» / пароль: Welcome2026
```

**Задание:** С помощью ИИ напишите объявление: коротко, понятно, с инструкцией.

**Формат:** Сообщение для чата 80-120 слов. **Бенчмарк:** 10 мин.

---

### ЗАДАНИЕ B-2: Поздравительное письмо от имени компании

**Вводные данные:**
```
Повод: юбилей сотрудницы — Елена Викторовна, главный бухгалтер, 50 лет
Стаж в компании: 8 лет (пришла одной из первых)
Что о ней можно сказать: держит на себе всю финансовую часть,
спокойная, надёжная, коллеги её уважают. Любит путешествия и кулинарию.
Формат: официальное поздравление от генерального + неформальное от коллектива
Подарок: сертификат в ресторан (10 000 ₽) — упомянуть в поздравлении
```

**Задание:** С помощью ИИ напишите два текста: официальный (для рамки на стену) и неформальный (для корпоративного чата).

**Формат:** 2 текста (100-150 слов каждый). **Бенчмарк:** 20 мин.

## УРОВЕНЬ: СРЕДНИЙ

### ЗАДАНИЕ M-1: Протокол совещания из заметок

**Вводные данные:**
```
Совещание: еженедельная планёрка руководства
Дата: 14.04.2026, 10:00-10:50
Участники: Андрей (CEO), Марина (коммерческий), Олег (IT), Катя (HR), я

Мои заметки:
«Андрей начал с продаж — Марина отчиталась, план на март 92%,
провал в сегменте B2C, B2B норм. Андрей недоволен, просит 
план по исправлению B2C к пятнице. Олег — сайт новый запустили, 
300 багов в первую неделю, сейчас 40, к концу апреля будет 0.
Андрей спросил про мобилку — Олег сказал Q3, раньше нереально.
Катя — 2 оффера отклонены, причина — зарплата ниже рынка.
Андрей просит подготовить аналитику рынка зарплат IT к среде.
Ещё обсудили корпоратив — 25 мая, бюджет 300 тыс, я координирую.
Следующая планёрка — понедельник 21.04 в 10:00.»
```

**Задание:** С помощью ИИ оформите протокол: повестка, ключевые тезисы, решения, таблица задач.

**Формат:** Протокол 1 страница. **Бенчмарк:** 30 мин.

### ЗАДАНИЕ M-2: Инструкция для новых сотрудников

**Вводные данные:**
```
Компания: IT-продуктовая, 70 сотрудников, Москва
Формат: гибрид (Пн-Ср офис, Чт-Пт удалённо)

Вам нужно составить «Гайд первого дня» — памятку, которую 
получает каждый новичок в первый рабочий день.

Что должно быть:
- Как пройти в офис (бизнес-центр «Кристалл», 5 этаж, код домофона 4527#)
- Где взять пропуск (ресепшен 1 этаж, с паспортом, в 9:00)
- Wi-Fi: CompanyNet / пароль: Welcome2026
- Рабочее место: назначает HR, спросить Катю (8-916-XXX)
- Обед: кухня на 5 этаже, микроволновка, кофемашина, 
  ближайшие кафе — «Чашка» (5 мин) и «Тандыр» (3 мин)
- Корпоративные инструменты: Slack (#general для объявлений),
  Jira (задачи), Confluence (документация), Google Workspace
- Парковка: подземная, 20 мест, бронирование через бота в Slack /parking
- Dress code: нет формального, но на встречи с клиентами — smart casual
- Больничный/отпуск: писать HR-боту в Slack + руководителю
- Экстренный контакт: Ирина (офис-менеджер), 8-926-XXX
```

**Задание:** С помощью ИИ составьте гайд первого дня: структурированный, дружелюбный, не перегруженный.

**Формат:** Документ 1.5-2 страницы. **Бенчмарк:** 45 мин.

### ЗАДАНИЕ M-3: Организация корпоративного мероприятия

**Вводные данные:**
```
Мероприятие: летний тимбилдинг
Дата: 15 июня (воскресенье)
Участники: 55 сотрудников + до 20 членов семей (дети тоже)
Бюджет: 350 000 ₽
Пожелания CEO: загород, активности, НЕ ресторан с караоке

Ограничения:
- До 1.5 ч езды от Москвы
- 3 сотрудника — вегетарианцы, 1 — аллергия на орехи
- Нужен трансфер (не все с машинами)
- Дети от 5 до 12 лет — нужны активности для них тоже
```

**Задание:** С помощью ИИ подготовьте:
1. 3 варианта площадок с плюсами/минусами (загородный отель, база отдыха, парк)
2. Программу дня (с тайм-слотами, от 10:00 до 20:00)
3. Смету (разбивка бюджета по категориям)
4. Чек-лист подготовки (что сделать за 2 мес / 1 мес / 1 нед до)

**Формат:** Документ (2-3 страницы). **Бенчмарк:** 60 мин.

## УРОВЕНЬ: ПРОДВИНУТЫЙ

### ЗАДАНИЕ A-1: Регламент внутренних коммуникаций

**Вводные данные:**
```
Компания: fintech-стартап, 90 сотрудников, 3 офиса (Москва, СПб, Казань) + удалённые

Проблемы:
- Всё обсуждается в личных сообщениях, решения теряются
- Нет единого стандарта: часть команды в Slack, часть в Telegram, часть в Teams
- Совещания по 2 часа без повестки и протокола
- Руководители получают 200+ сообщений/день и пропускают важное
- Новички первые 2 недели не понимают, куда и кому писать

Инструменты компании: Slack (основной), Google Meet (созвоны), 
Confluence (документация), Jira (задачи), Google Calendar

Что хочет CEO: «Один документ, который объясняет: что куда писать, 
как проводить совещания, как принимать решения. Чтобы любой 
новый сотрудник прочитал и сразу понял правила игры.»
```

**Задание:** С помощью ИИ разработайте регламент внутренних коммуникаций:
1. Каналы и их назначение (что куда: Slack-каналы, DM, email, созвоны)
2. Правила совещаний (когда нужна встреча, а когда нет; повестка; протокол; максимальная длительность)
3. Правила принятия решений (кто решает, как фиксируем, где хранится)
4. Правила для удалённых (доступность, часовые пояса, статусы)
5. Эскалация (что делать, если не ответили / не решили)
6. Этикет (нерабочее время, эмодзи, реакции, ответственность за прочтение)

**Формат:** Полный регламент (3-4 страницы). **Бенчмарк:** 120 мин.

### ЗАДАНИЕ A-2: Подготовка стратегической презентации для CEO

**Вводные данные:**
```
CEO выступает перед советом директоров с отчётом за 1-й квартал.
Нужно подготовить структуру и тексты для презентации.

Данные:
- Выручка Q1: 45 млн ₽ (+18% к Q1 2025)
- Клиентов: 1 200 (+30%)
- NPS: 72 (было 65)
- Текучесть: 12% (было 18%)
- Новых сотрудников: +15 (план был +20, не закрыли 5 IT-позиций)
- Запустили 2 новых продукта, 1 закрыли (не окупился)
- Главный риск: задержка платформы 2.0 (на 2 месяца от плана)

Аудитория: 5 членов совета директоров (инвесторы + основатели)
Стиль: данные → выводы → план. Без воды.
Длительность выступления: 20 мин + 10 мин Q&A
```

**Задание:** С помощью ИИ подготовьте:
1. Структуру презентации (10-14 слайдов) с описанием каждого
2. Текст для каждого слайда (заголовок + 3-5 буллетов + заметки спикера)
3. Список антикризисных ответов (5 «неудобных вопросов», которые могут задать, + ответы)

**Формат:** Полный документ (4-5 страниц). **Бенчмарк:** 90 мин.
