## УРОВЕНЬ: БАЗОВЫЙ

### ЗАДАНИЕ B-1: Пост для Telegram-канала

**Вводные данные:**
```
Компания: онлайн-школа английского для взрослых
Канал: Telegram, 5 000 подписчиков
ЦА: работающие люди 25-40 лет, хотят подтянуть English для карьеры
ToV: дружелюбный, мотивирующий, без менторства

Инфоповод: запускаем новый формат — 15-минутные уроки в обеденный 
перерыв (мини-курс «English Lunch»). Цена: 1 990 ₽/мес (8 уроков).
Старт: 1 мая. Первая неделя бесплатно.
```

**Задание:** С помощью ИИ напишите анонсный пост. Формат: зацепка + суть + CTA.

**Формат:** Пост 600-900 знаков с эмодзи. **Бенчмарк:** 15 мин.

**Вводные данные:**
```
Компания: сервис облачной телефонии для бизнеса

  Нажмите 1 для отдела продаж...» — написать 2-3 примера скриптов
```

### ЗАДАНИЕ M-3: Адаптация статьи в 4 формата

**Вводные данные:**
```
Исходная статья (ключевые тезисы):

1. Загрузка > 3 секунд (53% пользователей уходят)
2. Нет мобильной версии (70% трафика — мобильный)

4. Кнопка CTA не видна / непонятна
5. Нет социального доказательства (отзывы, кейсы)

Адаптировать в:
1. Telegram-пост (600-800 знаков, с эмодзи, вопрос к аудитории)
2. Email-рассылка (тема + 150 слов + CTA «Читать статью»)
3. Текст для карусели VK (8 слайдов: обложка + 5 ошибок + решение + CTA)
4. Короткое описание для Яндекс.Дзен (50 слов + интрига для клика)

ЦА: маркетологи и собственники МСБ. ToV: экспертный, но не академический.
```

**Задание:** С помощью ИИ создайте все 4 адаптации.

## УРОВЕНЬ: ПРОДВИНУТЫЙ

### ЗАДАНИЕ A-1: Контент-стратегия для нового продукта

**Вводные данные:**
```
Компания запускает новый продукт через 6 недель. Нужна контент-стратегия 
от pre-launch до post-launch.

Продукт: ИИ-ассистент для составления договоров (SaaS, B2B)
ЦА: юристы МСБ, собственники без штатного юриста
Цена: 2 900 ₽/мес (trial 7 дней)
Конкуренты: ничего точно такого в РФ, есть западные аналоги

Каналы: блог, Telegram (1 200 подп.), email (3 500 база), 
Хабр (для хайпа), YouTube (планируют завести)

Бюджет на контент: 150 000 ₽/мес (копирайтинг + дизайн)
Ресурсы: 1 контент-менеджер (вы) + дизайнер-фрилансер + CEO для экспертных 
комментариев

Этапы запуска:
- T-6 недель: pre-launch (awareness)
- T-2 недели: анонс + waitlist
- T=0: запуск + промо
- T+4 недели: post-launch (retention + growth)
```

**Задание:** С помощью ИИ разработайте полную контент-стратегию:
1. Цели и метрики для каждого этапа
2. Контент-план по неделям (10 недель: 6 pre + 4 post)
3. Форматы контента для каждого канала
4. Темы и заголовки ключевых материалов
5. Распределение бюджета
6. KPI: что измеряем и какие целевые значения

**Формат:** Стратегический документ (4-5 страниц). **Бенчмарк:** 180 мин.

### ЗАДАНИЕ A-2: Редакционная политика бренда

**Вводные данные:**
```
Компания: финтех-стартап, приложение для инвестиций для начинающих
ЦА: молодые специалисты 22-35 лет, первые инвестиции
Проблема: 3 автора пишут в разном стиле, нет единого голоса
CEO хочет: «чтобы наш контент звучал как умный друг, который 
объясняет финансы за кофе, а не как учебник по экономике»

Каналы: блог, TG, email, push-уведомления, экраны приложения

У вас есть:
- 5 примеров хороших текстов (которые нравились CEO и аудитории)
- 5 примеров плохих текстов (которые не зашли)
- Базовые ценности бренда: доступность, честность, без лишнего риска
```

**Задание:** С помощью ИИ разработайте редакционную политику (tone of voice guide):
1. Голос бренда (кто мы, как мы говорим, с кем сравнимы)
2. Принципы (3-5 правил с примерами «так» и «не так»)
3. Стоп-слова и стоп-паттерны (что НИКОГДА не пишем)
4. Адаптация по каналам (блог vs TG vs push — чем отличается тон)
5. Чек-лист для автора перед публикацией
6. 3 примера текстов «до и после» (плохой → хороший по вашим правилам)

**Формат:** Гайд (3-4 страницы). **Бенчмарк:** 120 мин.
