# ПУЛ ЗАДАНИЙ: МАРКЕТОЛОГ / DIGITAL

> **Профиль:** marketing | **Заданий:** 7 (2+3+2) | **Лимит:** 45 мин

---

## УРОВЕНЬ: БАЗОВЫЙ

### ЗАДАНИЕ B-1: Продающий пост из техописания

**Контекст:** Вы — маркетолог в компании, которая продаёт облачную CRM. Продакт-менеджер прислал техническое описание нового модуля. Нужно переделать его в пост для Telegram-канала.

**Вводные данные:**
```
Техническое описание:
«Модуль аналитики CRM v2.0 позволяет строить воронку продаж в реальном 
времени с детализацией до уровня каждого менеджера. Поддерживает 
настраиваемые дашборды, автоматическую выгрузку еженедельных отчётов 
в PDF и Excel, сравнение периодов, прогнозирование выполнения плана 
на основе текущей конверсии. Интеграция с Битрикс24, amoCRM, 1С.»

Telegram-канал: 2 500 подписчиков, ЦА — РОПы и собственники МСБ.
Tone of voice: экспертный, но живой. Без «инновационный», «уникальный».
```

**Задание:** С помощью ИИ напишите продающий пост для Telegram. Пост должен начинаться с боли ЦА, показывать решение через продукт и содержать CTA.

**Формат:** Пост 800-1200 знаков с эмодзи. **Бенчмарк:** 25 мин.

---

### ЗАДАНИЕ B-2: Email для реактивации «уснувших» клиентов

**Контекст:** Вы — маркетолог в SaaS-компании. Нужно вернуть пользователей, которые зарегистрировались, но не пользовались продуктом 30+ дней.

**Вводные данные:**
```
Продукт: сервис автоматизации email-рассылок, 990 ₽/мес
Сегмент: зарегистрировались, создали 1-2 рассылки, не заходили 30+ дней
Размер сегмента: 420 человек
Прошлые попытки: стандартное «Мы скучаем» — open rate 8%, конверсия 0.5%
```

**Задание:** С помощью ИИ напишите одно реактивационное письмо, которое будет лучше стандартного «Мы скучаем». Тема + прехедер + тело (150-200 слов) + CTA.

**Формат:** Готовое письмо. **Бенчмарк:** 20 мин.

## УРОВЕНЬ: СРЕДНИЙ

### ЗАДАНИЕ M-1: Welcome-цепочка из 3 писем

**Контекст:** Вы — email-маркетолог финтех-компании.

**Вводные данные (бриф):**
```
Продукт: онлайн-бухгалтерия для ИП и малого бизнеса, 1 900 ₽/мес
Триггер подписки: скачал чек-лист «10 ошибок в бухгалтерии МСБ»
ЦА: ИП и собственники малого бизнеса (25-50 лет)
Цель цепочки: познакомить с продуктом → показать ценность → trial
Trial: 14 дней бесплатно, не нужна карта

Интервалы: Письмо 1 — сразу, Письмо 2 — через 2 дня, Письмо 3 — через 5 дней
Тон: дружелюбно-профессиональный, не «впаривать»
```

**Задание:** С помощью ИИ создайте welcome-цепочку из 3 писем. Для каждого: тема (до 50 символов), прехедер, тело (150-250 слов), CTA.

**Формат:** 3 готовых письма. **Бенчмарк:** 90 мин.

---

### ЗАДАНИЕ M-2: Анализ конкурентов и формулировка УТП

**Контекст:** Руководитель попросил обновить позиционирование на сайте. Нужно проанализировать конкурентов и сформулировать УТП.

**Вводные данные:**
```
Ваш продукт: платформа для онлайн-записи клиентов (салоны красоты, клиники)
Цена: от 1 500 ₽/мес
Конкуренты: YCLIENTS, Dikidi, Арника
Ваши преимущества: бесплатная интеграция с WhatsApp, 
автоматические напоминания клиентам, виджет для сайта,
аналитика по мастерам. Техподдержка 24/7 (у конкурентов — по рабочим).
Слабость: менее известны, нет мобильного приложения (в разработке).
```

**Задание:** С помощью ИИ подготовьте:
1. Таблицу сравнения с 3 конкурентами (по 5-7 критериям)
2. 3 варианта УТП (заголовок + подзаголовок для главной страницы)
3. Рекомендацию: какое УТП выбрать и почему

**Формат:** Аналитический документ (1-2 страницы). **Бенчмарк:** 60 мин.

### ЗАДАНИЕ M-3: Описания товаров из характеристик

**Контекст:** Интернет-магазин добавляет новую линейку. Нужно быстро написать описания.

**Вводные данные:**
```
Магазин: товары для дома и кухни, средний чек 3 500 ₽
ЦА: женщины 28-45 лет, ценят качество и дизайн
Тон: тёплый, практичный, без «премиальный» и «эксклюзивный»

Товар 1: Набор керамических ножей (3 шт.)
Характеристики: лезвие — циркониевая керамика, длина 10/13/15 см, 
вес набора 280 г, ручка — бамбук, подставка в комплекте.

Товар 2: Электрическая кофемолка
Характеристики: мощность 200 Вт, ёмкость 80 г, 
нержавеющая сталь, импульсный режим, шнур 1.2 м.

Товар 3: Силиконовый набор для выпечки (6 предметов)
Характеристики: силикон пищевой (-40 до +230°C), 
антипригарный, формы: кекс, маффин, хлеб, бублик, пончик, тарталетка.
```

**Задание:** С помощью ИИ напишите описания для 3 товаров. Каждое: заголовок + описание 100-150 слов + 3 ключевых преимущества.

**Формат:** 3 готовых описания. **Бенчмарк:** 45 мин.

## УРОВЕНЬ: ПРОДВИНУТЫЙ

### ЗАДАНИЕ A-1: Контент-план на месяц

**Контекст:** Вам нужно спланировать весь контент на май.

**Вводные данные:**
```
Компания: B2B SaaS, CRM для малого бизнеса
ЦА: собственники и РОПы компаний 10-50 чел.
Каналы: Telegram (2 500 подп.), email (база 8 000), блог на сайте
ToV: экспертный, но живой

Ключевые события мая:
1. Релиз модуля аналитики (5 мая)
2. Кейс клиента: «ПромТех» увеличил конверсию на 40%
3. Конференция «Digital Retail» (20 мая, стенд + выступление)
4. Майские праздники (1-3, 9 мая)

Бюджет на промо: 80 000 ₽/мес (Telegram Ads + VK)
Баланс контента: 40% экспертный, 30% продуктовый, 20% кейсы, 10% лёгкий
```

**Задание:** С помощью ИИ создайте полный контент-план на май:
- Для каждой недели: 3 поста Telegram + 1 email + 1 статья блога
- Для каждой единицы: тема, формат, цель, ключевые тезисы
- Отдельно: план продвижения релиза и конференции

**Формат:** Таблица контент-плана + 2 мини-стратегии. **Бенчмарк:** 180 мин.

### ЗАДАНИЕ A-2: Лендинг — структура + тексты

**Контекст:** Нужен лендинг для нового продукта. Дизайнер ждёт от вас структуру и тексты.

**Вводные данные:**
```
Продукт: мобильное приложение для учёта личных финансов
ЦА: молодые специалисты 23-35 лет, доход 80-150 тыс ₽/мес
Цена: бесплатно (базовое), 199 ₽/мес (Pro)
Конкурент: дзен-мани, CoinKeeper
Ключевые фичи: 
- Автоматический учёт из SMS банков
- ИИ-подсказки «Где можно сэкономить»
- Совместный бюджет (для пар)
- Цели накопления с прогнозом
Цель лендинга: скачивание из App Store / Google Play
```

**Задание:** С помощью ИИ подготовьте:
1. Структуру лендинга (10-12 экранов/блоков)
2. Тексты для каждого блока (заголовок + подзаголовок + тело)
3. 3 варианта hero-секции (главный экран)
4. Meta-данные (title, description для SEO)

**Формат:** Документ с полной структурой и текстами. **Бенчмарк:** 120 мин.
