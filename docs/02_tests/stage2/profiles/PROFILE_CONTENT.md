## 2. ТОП-3 ЗАДАЧИ ДЛЯ АВТОМАТИЗАЦИИ

### 2.1 Генерация текстового контента (посты, описания, рассылки)
- **Почему #1:** Ежедневный объём, текстовый формат, чёткий ToV = идеально для ИИ
- **Что делает ИИ:** Первый драфт, вариации, адаптация под каналы, массовая генерация описаний
- **Что остаётся человеку:** Редактура, фактчекинг, уникальность, «голос бренда»
- **Ускорение-ориентир:** 5-15x

### 2.2 Статьи и лонгриды
- **Почему #2:** Самая трудоёмкая задача, ИИ ускоряет структуру и черновик
- **Что делает ИИ:** Планирует структуру, пишет черновик по тезисам, подбирает аргументы
- **Что остаётся человеку:** Экспертиза, уникальный опыт, голос автора, SEO-оптимизация вручную
- **Ускорение-ориентир:** 3-5x

### 2.3 Адаптация контента под каналы
- **Почему #3:** Один материал → 5 форматов (блог → пост TG → email → карусель → шорт)
- **Что делает ИИ:** Переформатирует с учётом ограничений канала (длина, тон, CTA)
- **Что остаётся человеку:** Визуал, финальная подгонка
- **Ускорение-ориентир:** 5-10x

## 3. ЭТАЛОННЫЕ ПРОМПТЫ-ОБРАЗЦЫ

### Эталон 1: SEO-статья для блога

```
Ты — контент-маркетолог B2B SaaS-компании (CRM для МСБ).

Задача: напиши SEO-статью для блога.

Тема: «Как выбрать CRM для малого бизнеса в 2026 году»
Целевой ключ: «CRM для малого бизнеса» (частотность 4 800/мес)
LSI-ключи: выбор CRM, CRM система для малого бизнеса, 
лучшая CRM для малого, внедрение CRM

Целевая аудитория: собственники компаний 5-30 чел., 
не технические, решают сами

Структура статьи:
1. Зацепка: боль ЦА (потерянные заявки, Excel, хаос в продажах)
2. Зачем CRM малому бизнесу (3 аргумента с цифрами)
3. 7 критериев выбора CRM (каждый — подзаголовок H2)
4. Сравнение 5 популярных CRM (таблица)
5. Чек-лист выбора (скачиваемый — лид-магнит)
6. Заключение + CTA

Тон: экспертный, но без зауми. Без «в современном мире» и 
«не секрет, что». С конкретными примерами.
Длина: 2000-2500 слов. H1/H2 с ключами, но без переспама.

Важно: не упоминай конкурентов по имени негативно.
Наш продукт можно упомянуть 1-2 раза, но статья — не реклама.
```

### Эталон 2: Пакетная генерация описаний товаров

```
Ты — копирайтер интернет-магазина бытовой техники.

Задача: напиши описания для 3 товаров.

Общие параметры:
- ЦА: покупатели 25-45 лет, выбирают для дома/квартиры
- Тон: практичный, тёплый, без маркетинговых клише
- Структура каждого: заголовок (с ключевым преимуществом) 
  + описание (100-150 слов) + 4 буллета «Почему выбирают»
- SEO: естественное вхождение названия + категории товара

Товар 1:
Название: Робот-пылесос XClean Pro 3000
Характеристики: мощность 3000 Па, LiDAR навигация, влажная уборка,

Товар 2:
Название: Увлажнитель воздуха AirMist S5
Характеристики: ультразвуковой, объём 5 л, площадь до 40 м²,
гигростат встроенный, ароматерапия, ночной режим (22 дБ),
расход 350 мл/ч. Цена: 4 990 ₽.

Товар 3:
Название: Электрогриль DualGrill 2200
Характеристики: 2200 Вт, раскрытие 180°, антипригарное покрытие,
регулировка температуры (180-230°C), поддон для жира, 
сменные пластины (гриль + вафельница). Цена: 7 490 ₽.
```

### Эталон 3: Адаптация статьи в 3 формата

```
Вот статья из нашего блога (ключевые тезисы):

Тема: «5 ошибок при внедрении CRM»
1. Внедряют без подготовки данных (мусор из Excel → мусор в CRM)
2. Не обучают сотрудников (купили — и ждут чуда)
3. Не назначают ответственного за CRM
4. Пытаются автоматизировать хаос (сначала процессы, потом CRM)
5. Выбирают по цене, а не по задачам

Адаптируй в 3 формата:

1. Пост для Telegram (600-800 знаков, с эмодзи, заканчивается 
   вопросом к аудитории, ссылка на полную статью)

2. Email-дайджест (тема + прехедер + 150 слов тела, 
   CTA: «Читать полную статью»)

3. Карусель для VK / Instagram (8 слайдов):
   - Слайд 1: цепляющий заголовок
   - Слайды 2-6: по 1 ошибке (заголовок + 1-2 предложения)
   - Слайд 7: «Что делать вместо этого» (3 буллета)
   - Слайд 8: CTA + контакты

Тон: экспертный, но живой. ЦА: РОПы и собственники МСБ.
```

## 4. ТИПИЧНЫЕ ОШИБКИ

| # | Ошибка | Пример | Что делать |
|---|--------|--------|-----------|
| 1 | Публикация без редактуры | Выдал ИИ-текст как финальный | Всегда редактировать: убрать клише, добавить голос |
| 2 | Нет ToV бренда | Каждый текст звучит по-разному | Включить в промпт: примеры хороших текстов бренда |
| 3 | SEO-переспам | «CRM для бизнеса — лучшая CRM система для бизнеса» | Указать: естественное вхождение, не более N раз |
| 4 | Фактические ошибки | ИИ написал «рынок CRM в РФ — $5 млрд» (выдумка) | Проверять все цифры, статистику, цитаты |
| 5 | Один формат на все каналы | Длинная статья в пост для TG | Указать канал и его ограничения в промпте |
| 6 | Нет CTA | Текст без призыва к действию | Всегда указывать: цель текста и CTA |
| 7 | Клише ИИ | «В современном мире», «Не секрет, что» | Добавить в промпт стоп-слова |

## 6. ПРОФИЛЬНЫЕ КРИТЕРИИ КАЧЕСТВА РЕЗУЛЬТАТА

### Для задачи «Статья / лонгрид»:
- Структура логична (от боли к решению): 0-2
- SEO-оптимизация (заголовки, ключи, мета): 0-2
- Тон соответствует ЦА: 0-2
- Нет клише и «воды»: 0-2
- Есть практическая ценность (не пересказ очевидного): 0-2
- **Максимум: 10**

### Для задачи «Описания товаров»:
- Адаптировано под ЦА (не спецификация, а польза): 0-3
- Структура единообразна: 0-2
- Есть ключевое преимущество в заголовке: 0-2
- Тон бренда соблюдён: 0-2
- Длина соответствует ТЗ: 0-1
- **Максимум: 10**

### Для задачи «Мультиформат»:
- Каждый формат адаптирован под канал: 0-3
- Ключевое сообщение сохранено во всех: 0-2
- CTA есть в каждом формате: 0-2
- Длина каждого формата адекватна: 0-2
- Тон единый: 0-1
- **Максимум: 10**
