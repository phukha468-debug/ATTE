## 2. ТОП-3 ЗАДАЧИ ДЛЯ АВТОМАТИЗАЦИИ

### 2.1 Контент: посты + email + описания
- **Почему #1:** Ежедневная текстовая рутина, высочайший потенциал ИИ
- **Что делает ИИ:** Генерирует тексты по ToV бренда, адаптирует под канал, создаёт вариации
- **Что остаётся человеку:** Стратегия, финальная редактура, визуал
- **Ускорение-ориентир:** 5-15x в зависимости от задачи

### 2.2 Контент-план и стратегия
- **Почему #2:** Высокая трудоёмкость, ИИ отлично структурирует
- **Что делает ИИ:** Генерирует темы, распределяет по форматам/каналам, учитывает сезонность
- **Что остаётся человеку:** Приоритизация, бюджеты, согласование
- **Ускорение-ориентир:** 5-10x

### 2.3 Анализ (конкуренты, SEO, отчёты)
- **Почему #3:** Много ручной обработки данных, ИИ хорошо структурирует выводы
- **Что делает ИИ:** Структурирует данные, выделяет тренды, формирует выводы
- **Что остаётся человеку:** Проверка данных, стратегические решения
- **Ускорение-ориентир:** 3-5x

## 3. ЭТАЛОННЫЕ ПРОМПТЫ-ОБРАЗЦЫ

### Эталон 1: Контент-план

```
Ты — digital-маркетолог B2B SaaS-компании с 5-летним опытом.

Задача: составь контент-план на апрель 2026.

Контекст компании:
- Продукт: облачная CRM для малого бизнеса
- ЦА: собственники и РОПы компаний 10-50 чел.
- Tone of voice: экспертный, но живой (без канцеляризмов)
- Каналы: Telegram (2500 подписчиков), email (база 8000), блог на сайте

Ключевые инфоповоды апреля:
1. Запуск модуля аналитики (релиз 10.04)
2. Кейс: клиент увеличил конверсию на 40%
3. Участие в конференции «Digital Retail» (22.04)

Для каждой недели дай:
- 3 поста Telegram (тема + формат: текст/карусель/видео)
- 1 email (тема + цель: вовлечение/продажа/обучение)
- 1 статья блога (тема + ключевые тезисы)

Формат: таблица. Учти распределение: 40% экспертный контент, 30% продуктовый, 20% кейсы, 10% развлекательный.
```

### Эталон 2: Продающий пост из техописания

```
Перепиши техническое описание продукта в продающий пост для Telegram.

Техническое описание:
«Модуль аналитики CRM позволяет строить воронку продаж в реальном 
времени, отслеживать конверсию на каждом этапе, выгружать отчёты 
в формате PDF и Excel, настраивать дашборды с произвольными метриками.»

Параметры поста:
- ЦА: руководители отделов продаж (28-45 лет)
- Тон: живой, с конкретными цифрами, без «инновационный» и «уникальный»
- Длина: 800-1200 знаков
- Включить: 1 конкретную боль ЦА → как продукт решает → CTA
- Эмодзи: 3-5, уместные
- CTA: «Попробуйте бесплатно 14 дней → ссылка»
```

### Эталон 3: Welcome-цепочка писем

```
Ты — email-маркетолог финтех-компании для МСБ.

Задача: создай welcome-цепочку из 3 писем для новых подписчиков.

Контекст:
- Продукт: сервис онлайн-бухгалтерии, подписка 1 900 ₽/мес
- Подписчик: оставил email на лендинге, скачал чек-лист «10 ошибок в бухгалтерии МСБ»
- Цель цепочки: познакомить → показать ценность → конвертировать в trial

Для каждого письма дай:
- Тема (до 50 символов, без капса и спам-слов)
- Прехедер (до 80 символов)
- Тело (150-250 слов)
- CTA (один, конкретный)

Интервал: Письмо 1 — сразу, Письмо 2 — через 2 дня, Письмо 3 — через 5 дней.
Тон: дружелюбно-профессиональный. Не «впаривать», а помогать.
```

## 4. ТИПИЧНЫЕ ОШИБКИ МАРКЕТОЛОГОВ ПРИ РАБОТЕ С ИИ

| # | Ошибка | Пример | Что делать |
|---|--------|--------|-----------|
| 1 | Нет ToV | «Напиши пост» без тона бренда | Указать: тон, ЦА, примеры текстов бренда |
| 2 | Нет ЦА | Generic текст «для всех» | Указать: возраст, должность, боль, мотивацию |
| 3 | Копипаст без адаптации | Один текст на все каналы | Указать канал + его ограничения (длина, формат) |
| 4 | Нет метрик/цифр | «Увеличили продажи» | Просить конкретику: «на 40% за 3 месяца» |
| 5 | Доверие к данным ИИ | ИИ выдал «средний CTR email 45%» | Проверять все цифры и статистику вручную |
| 6 | Нет структуры CTA | Пост без призыва к действию | Указать: один CTA, формат, куда ведёт |
| 7 | Не сохраняет промпты | Каждый раз пишет заново | Создать библиотеку промптов по задачам |

## 6. ПРОФИЛЬНЫЕ КРИТЕРИИ КАЧЕСТВА РЕЗУЛЬТАТА

### Для задачи «Контент-план»:
- Покрыты все каналы из брифа: 0-2
- Учтены инфоповоды/релизы: 0-2
- Баланс типов контента (не всё продающее): 0-2
- Форматы разнообразны (не только текст): 0-2
- Реалистичность (можно реально выпустить за месяц): 0-2
- **Максимум: 10**

### Для задачи «Пост / email»:
- Адаптация под ЦА (не generic): 0-3
- Есть конкретная боль → решение: 0-2
- Тон соответствует брифу: 0-2
- Есть CTA: 0-1
- Длина соответствует каналу: 0-2
- **Максимум: 10**

### Для задачи «Анализ конкурентов»:
- Структура (не поток текста): 0-2
- Конкретные выводы, а не пересказ: 0-3
- Есть actionable рекомендации: 0-3
- Адаптировано под контекст компании: 0-2
- **Максимум: 10**
