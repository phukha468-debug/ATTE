## 2. ТОП-3 ЗАДАЧИ ДЛЯ АВТОМАТИЗАЦИИ

### 2.1 Разъяснения и коммуникации (сотрудникам, руководителю, контрагентам)
- **Почему #1:** Ежедневная текстовая рутина, ИИ превосходно справляется с трансляцией бухгалтерского языка в понятный
- **Что делает ИИ:** Переводит «оплатите НДС по ст. 174 НК» в «прикрепите чек и напишите назначение — я оформлю»
- **Что остаётся человеку:** Проверка нормативных ссылок, конкретные суммы и реквизиты
- **Ускорение-ориентир:** 5-10x

### 2.2 Регламенты, памятки, инструкции
- **Почему #2:** Одна хорошая памятка снижает 50% повторяющихся вопросов от сотрудников
- **Что делает ИИ:** Структурирует процесс, формулирует инструкцию в понятной форме, создаёт варианты для разных ролей
- **Что остаётся человеку:** Проверка соответствия нормам, актуализация ссылок на законодательство
- **Ускорение-ориентир:** 5-8x

### 2.3 Аналитические записки для руководителя
- **Почему #3:** Данные берутся из 1С / Excel, но текстовая интерпретация — ручная работа
- **Что делает ИИ:** По загруженным цифрам формирует структурированные выводы, подсвечивает отклонения
- **Что остаётся человеку:** Проверка расчётов, контекст (почему так), рекомендации
- **Ускорение-ориентир:** 5-8x

## 3. ЭТАЛОННЫЕ ПРОМПТЫ-ОБРАЗЦЫ

### Эталон 1: Памятка для сотрудников по авансовым отчётам

```
Ты — главный бухгалтер в IT-компании (50 сотрудников, упрощёнка).

Задача: составь памятку для сотрудников по оформлению авансовых отчётов
после командировок.

Контекст:
- Аудитория: менеджеры, программисты, маркетологи — НЕ бухгалтеры
- Уровень подготовки: большинство не понимает, что такое «первичный документ»
- Частые ошибки: забывают чеки, неправильно оформляют, путают суточные и расходы
- Срок сдачи: 3 рабочих дня после возвращения (по нашему регламенту)

Структура памятки:
1. Что считается командировкой (коротко)
2. Что взять с собой в командировку (чек-лист)
3. Какие документы собрать во время поездки (что, зачем, образец)
4. Как заполнить авансовый отчёт (пошагово, с примерами)
5. Частые ошибки и как их избежать
6. Куда обращаться при вопросах

Тон: дружелюбный, человеческий. Сотрудник не должен чувствовать себя 
тупым, если не знает разницу между «суточными» и «возмещением расходов».
Без «в соответствии с п. 3 ст. 168 ТК РФ» — это отпугнёт.
Длина: 1-1.5 страницы. Добавь эмодзи-маркеры (✅ / ⚠️ / 📎) для структуры.
```

### Эталон 2: Ответ контрагенту на запрос документов

```
Напиши письмо контрагенту в ответ на запрос документов для сверки.

Контекст:
- Наша компания: ООО «РазработкаПлюс» (продавец услуг)
- Контрагент: ООО «Стройтех» (покупатель)
- Запрос контрагента: «Просим прислать акты выполненных работ и 
  счета-фактуры за период март-апрель 2026 для квартальной сверки»
- Период их запроса: март-апрель
- Что у нас есть: все акты (4 штуки), все счета-фактуры (4 штуки)
- Все документы подписаны с двух сторон
- Задолженность контрагента перед нами: 450 000 ₽ (3 неоплаченных счёта)

Задача письма:
1. Ответить на запрос (приложить документы)
2. МЯГКО напомнить о задолженности (не в тоне «когда заплатите», 
   а «обратите внимание на открытые позиции»)
3. Предложить провести сверку и подписать акт сверки

Тон: деловой, доброжелательный. Длина: 100-150 слов.
```

### Эталон 3: Пояснительная записка к отчёту

```
Ты — финансовый бухгалтер. Руководитель попросил подготовить 
пояснительную записку к отчёту по расходам за март.

Цифры из отчёта (все суммы в ₽, без НДС):

| Статья расходов      | Февраль   | Март      | Изменение |
| Аренда офиса         | 180 000   | 180 000   | 0%        |
| ФОТ + страховые      | 2 450 000 | 2 680 000 | +9.4%     |
| Реклама              | 450 000   | 720 000   | +60%      |
| ИТ-сервисы           | 85 000    | 92 000    | +8.2%     |
| Командировки         | 45 000    | 180 000   | +300%     |
| Канцтовары           | 15 000    | 18 000    | +20%      |
| Прочие               | 60 000    | 75 000    | +25%      |
| ИТОГО                | 3 285 000 | 3 945 000 | +20%      |

Что я знаю о контексте:
- В марте приняли 3 новых сотрудника (рост ФОТ объясним)
- Запустили крупную рекламную кампанию к весеннему сезону
- Команда ездила на конференцию «Digital Retail» (Москва, 5 чел., 4 дня)
- Аномалия по канцтоварам — не понимаю причину

Напиши пояснительную записку (1 страница):
- Ключевые изменения с объяснением
- Что требует внимания руководителя
- Что требует дополнительного анализа (канцтовары)
- Рекомендации (если есть)

Тон: деловой, без эмоций. Руководитель — собственник, не бухгалтер.
Избегать «расходы увеличились в связи с» — это канцелярщина.
```

## 4. ТИПИЧНЫЕ ОШИБКИ БУХГАЛТЕРОВ ПРИ РАБОТЕ С ИИ

| # | Ошибка | Пример | Что делать |
|---|--------|--------|-----------|
| 1 | Загрузка конфиденциальных данных | Копирует выписки банка с реквизитами в бесплатный ChatGPT | Корпоративный API + обезличивание |
| 2 | Доверие к юридическим нормам | ИИ выдал «срок хранения первички — 3 года» (на самом деле 5) | ВСЕГДА проверять ссылки на НК/ТК/ФЗ |
| 3 | Расчёты от ИИ | Просит посчитать зарплату / НДФЛ | ИИ не считает — работа в 1С/Excel с формулами |
| 4 | Бухгалтерский язык в промпте | «Составь пояснение к сч. 60 по договору 2023/04» | Добавить контекст на человеческом языке |
| 5 | Без контекста компании | Generic памятка без учёта системы налогообложения | Указать: УСН / ОСНО, отрасль, размер |
| 6 | Копипаст писем ФНС | Принимает формулировки ИИ в ответе налоговой | Все официальные ответы ФНС — вручную, ИИ только черновик |
| 7 | Переоценка возможностей | «ИИ заполнит декларацию» | ИИ помогает с пояснениями, декларации — в 1С |

## 5. СВЯЗКА С ЭТАПОМ 1

```
ЕСЛИ балл_этап1 >= 85% И промптинг >= 75%:
    → Продвинутый (пояснительная к отчёту / регламент с аудитом)
ЕСЛИ балл_этап1 >= 75% И промптинг >= 60%:
    → Средний (памятка сотрудникам / сверка с контрагентом)
ЕСЛИ балл_этап1 < 75% ИЛИ промптинг < 60%:
    → Базовый (одно письмо / один ответ сотруднику)
```

## 6. ПРОФИЛЬНЫЕ КРИТЕРИИ КАЧЕСТВА РЕЗУЛЬТАТА

### Для задачи «Памятка / инструкция»:
- Понятна НЕ-бухгалтеру (без жаргона): 0-3
- Структурирована (пошагово): 0-2
- Покрывает частые ошибки: 0-2
- Есть практические примеры: 0-2
- Указаны контакты для вопросов: 0-1
- **Максимум: 10**

### Для задачи «Письмо контрагенту»:
- Деловой тон, без фамильярности: 0-2
- Чёткая структура (запрос → ответ → дальнейшие шаги): 0-3
- Дипломатичность при напоминании о задолженности: 0-2
- Правильные реквизиты и цифры: 0-2
- Нет корпоративного языка («доводим до вашего сведения»): 0-1
- **Максимум: 10**

### Для задачи «Пояснительная записка»:
- Ключевые изменения объяснены: 0-3
- Подсвечены аномалии / требующие внимания: 0-3
- Рекомендации конкретны: 0-2
- Язык понятен не-бухгалтеру: 0-2
- **Максимум: 10**
