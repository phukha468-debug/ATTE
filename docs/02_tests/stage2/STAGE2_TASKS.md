# ЗАДАНИЯ ЭТАПА 2: НАВЫКИ В ДЕЛЕ (ИИ-Симулятор)

> **Версия:** 3.0
> **Суть:** Сотрудник решает реальную задачу из своей профессии с помощью ИИ.
> Система оценивает не теорию, а практическое умение экономить время.

---

## АРХИТЕКТУРА ОЦЕНКИ

### Dual-AI система
- **ИИ-Ассистент (Sandbox):** нейтральный помощник, выполняет запросы, не подсказывает
- **ИИ-Судья (Judge):** оценивает постфактум по 5 измерениям

### Модульная оценка (Core Skill + Profile Module)
1. **Core Skill** оценивает универсальные навыки: Промптинг (30%), Итеративность (20%), Ускорение (20%), Ограничения (10%)
2. **Profile Module** оценивает специфическое: Качество результата (20%) по критериям конкретной профессии

### Динамическая сложность (по результатам Этапа 1)

| Результат Этапа 1 | Уровень задания | Характер |
|-------------------|-----------------|----------|
| < 75% или промптинг < 60% | Базовый | Одна задача, 1-2 промпта |
| ≥ 75%, промптинг ≥ 60% | Средний | 2-3 итерации, развёрнутый бриф |
| ≥ 85%, промптинг ≥ 75% | Продвинутый | Сложный бриф, пайплайн из шагов |

### Связка с Этапом 1
Задание подбирается близко к задачам из карты рутины. Если продажник написал, что тратит 2 часа на КП — он получит задачу на составление КП.

### Правила для всех профилей
1. Сотрудник работает в Sandbox (чат с ИИ). Все промпты записываются.
2. Лимит времени: 45 минут.
3. Можно отправлять сколько угодно промптов и дорабатывать.
4. Финальный результат загружается как ответ.
5. После завершения — автоматическая оценка Judge + обратная связь.
6. Задания ротируются — сотрудники из одного отдела получают разные.
