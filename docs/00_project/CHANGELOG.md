## [v3.0] — Текущая версия (Апрель 2026)
**Ключевое изменение: Переход к трехуровневой архитектуре и Dual-AI.**
- **Добавлено:** Этап 2 (ИИ-Симулятор) с реальными рабочими задачами и таймером.
- **Добавлено:** Архитектура Dual-AI (ИИ-Ассистент для помощи + LLM-Судья для оценки постфактум).
- **Добавлено:** Модульная система оценки (1 Core Skill + 20 Profile Modules).
- **Добавлено:** Этап 3 (Микро-проекты) — внедрение ИИ в реальную работу с оценкой по AI Value Rubric (20 баллов).
- **Изменено:** Обратная связь теперь включает разбор конкретных промптов и валидацию потенциала экономии.

## [v2.0] — Автоматизация оценки
- **Добавлено:** Внедрение LLM-Судьи (Judge v1) для автоматической оценки открытых ответов Этапа 1.
- **Добавлено:** 4 домена оценки (Промптинг, Рутина, Ограничения, Право).
- **Изменено:** Отказ от ручной проверки тестов экспертами.

## [v1.0] — Базовая версия (MVP)
- **Создано:** Концепция диагностики ИИ-компетенций (15 базовых вопросов).
- **Создано:** Telegram-бот для сбора ответов.
- **Создано:** Формирование первой Карты рутины (сбор данных текстом).
